📝# 📝 Python To-Do List

A beginner-friendly console-based To-Do List application built using Python to strengthen core programming concepts through a real-world project.

This project allows users to manage daily tasks with features like task creation, searching, editing, deadlines, file storage, and status tracking. It is being developed step by step while focusing on clean code, beginner-friendly implementation, and practical problem-solving.

---

## ✨ Features

- ➕ Add new tasks
- 📋 View all tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 🔍 Search tasks (case-insensitive)
- ✏️ Edit existing tasks
- 📅 Optional deadlines
- ⏳ Remaining days calculation
- ⚠️ Overdue task detection
- 💾 Automatic saving using file handling
- ✔️ Input validation and exception handling

---

## 🛠️ Concepts Used

- Python Functions
- Lists & Dictionaries
- Loops
- Conditional Statements
- File Handling
- Exception Handling
- Date & Time (`datetime`)
- String Manipulation
- Input Validation

---

## 📂 Task Storage

Tasks are stored in a text file (`tasks.txt`) using the format:

Task Name | Status | Deadline

This allows all tasks to persist even after closing the program.
